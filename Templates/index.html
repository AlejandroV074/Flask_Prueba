Página principal

Para la instalar la librería "Flask", debemos hacerlo con el comando pip. 

pip install flask

Para arrancar la aplicación, debe usarse el siguiente código: 

python -m falsk run

Con este código al momento de editar el programa, no hará ningún cambio en tiempo real,
esto quiere decir que apagar el servidor, guardar el cambio y refrescar el navegador.


Para mirar el cambio en tiempo real debemos usar el siguiente código: 

if__name__=="__main__": 
    app.run(debug=True)

Y para ejecutarlo, debemos escribir 

python -m app run
